# Digital Business Card App

## Description
This project is a digital business card application developed using Kotlin. The app was created to practice mobile application development concepts while designing a simple and professional way to display personal information digitally.

The application presents contact information, professional details, and a clean user interface that can be used as a digital alternative to a traditional business card.

## Features
- Digital business card layout
- Displays personal and professional information
- Clean user interface design
- Mobile application interface
- Organized screen layout
- User-friendly design

## Technologies Used
- Kotlin
- Android Studio
- Android SDK
- XML Layout Design
- Android Application Components

## Skills Practiced
- Kotlin programming fundamentals
- Android app development
- Designing user interfaces
- Working with layouts and components
- Managing app resources
- Testing and debugging mobile applications

## How It Works
1. The user opens the application.
2. The app displays the digital business card.
3. Users can view the provided contact and professional information.

## Purpose
The goal of this project was to learn the fundamentals of Kotlin and Android development while creating a practical mobile application with a real-world use case.
